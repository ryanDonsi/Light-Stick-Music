## 🎯 개발 개요

**목표**: Dual ODF와 DBN HMM을 통해 K-POP 특화 정확도 향상 (+4.2%p)  
**전략**: 링버퍼 기반 메모리 관리로 모든 스마트폰 지원 (2GB RAM 이상)

---

## 💡 핵심 기술 4가지

### 1. Dual ODF 아키텍처
- odfTempo + odfTrack로 다각도 분석
- K-POP 정확도: 83.8% → 86.8%

### 2. 스트리밍 메모리 관리
- 링버퍼 2048 샘플 (32KB 고정)
- 메모리 99.9% 절감

### 3. DBN HMM 기반 BPM 추정
- 120 BPM 중심 확률 모델
- BPM 안정성 향상

### 4. Dynamic Programming + 910ms Guard
- 정교한 다운비트 감지
- 반박자 보정 강화
