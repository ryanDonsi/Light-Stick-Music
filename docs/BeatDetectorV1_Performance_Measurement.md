## 🔄 BeatDetectorV1 알고리즘 특성

**처리 방식**:
- PCM 입력 기반 (envelope 동시 추출)
- IIR 3밴드 Onset Detection Function (ODF)
- 자기상관 (Autocorrelation) + Log-Normal Prior
- 동적 프로그래밍 (DP) 기반 위상 추적 (선택사항)

**특징**:
- 중간 정확도, 중간 속도
- 일반적인 음악 (발라드~댄스) 최적화
- 낮은 메모리 사용량
- PCM 버퍼 크기 제한 (OOM 방지)
