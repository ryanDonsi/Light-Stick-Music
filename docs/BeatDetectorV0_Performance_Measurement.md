## 🔄 BeatDetectorV0 알고리즘 특성

**처리 방식**:
- PCM Envelope 추출 기반
- 간단한 onset detection
- 저주파 필터링으로 노이즈 제거
- BPM 추정 및 비트 배치

**특징**:
- 낮은 정확도, 매우 빠른 속도
- 단순 음악 (리듬감 명확한 댄스, 팝) 최적화
- 극도로 낮은 메모리 사용량
- 빠른 프로토타입 및 테스트용 최적
