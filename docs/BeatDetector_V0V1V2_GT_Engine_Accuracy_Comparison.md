**분석 의견**:
- **Precision (정확성)**: 예측된 비트 중 실제 비트의 비율. 높을수록 거짓양성(false positive) 적음
- **Recall (재현율)**: 실제 비트 중 맞게 예측된 비율. 높을수록 거짓음성(false negative) 적음
- V0는 상대적으로 precision이 낮은 경향 → 과도하게 비트를 검출
- V1과 V2는 precision과 recall의 균형이 더 나음 → 더 정확한 비트 감지
- K-POP Pop과 K-POP Dance에서 최고 성능 → 규칙적인 비트 패턴에 적합
- BPM 오차율은 버전이 올라갈수록 감소 → 더 정확한 템포 추정
- madmom 기준 정확도가 Beat Transformer보다 높아 신뢰도가 더 높음
