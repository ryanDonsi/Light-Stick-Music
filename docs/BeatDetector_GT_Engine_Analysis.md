## 📊 엔진별 역할 및 특성

### Beat Transformer
- **특징**: 딥러닝 기반 비트 감지
- **장점**: 복잡한 음악, 다양한 장르에 강함
- **용도**: 일반적인 비트 감지, 프로덕션 환경

### librosa
- **특징**: Python 기반 음악 정보 검색 라이브러리
- **장점**: 가벼움, 빠른 계산
- **용도**: 프로토타입, 교육용

### madmom
- **특징**: 음악 정보 검색 라이브러리 (높은 정확도)
- **장점**: 가장 정확한 비트 감지
- **용도**: 고정확도 필요, 연구 목적
