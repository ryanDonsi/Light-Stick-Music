- *처리 속도는 하드웨어 사양(CPU/GPU), 곡의 길이, GPU 드라이버 최적화에 따라 크게 달라집니다
- madmom: CPU만 사용 시 5분 곡 ~20초, GPU 권장 시 더 빠름
- Beat Transformer: GPU 필수, 5분 곡 ~30초 (고사양 GPU 사용 시 더 빠름)
- librosa: CPU만 필요, 5분 곡 ~2초

## 주요 특징

### madmom (2016, ISMIR)
- **장점**: 안정적 성능, 다양한 모델 앙상블, K-POP 음악에 우수
- **단점**: Beat Transformer보다 2.9%p 낮은 정확도
- **추천**: 프로덕션 환경, 정박자 음악 분석

### Beat Transformer (2023, Meta AI)
- **장점**: 최고 정확도(88.5%), 최신 Transformer 기술
- **단점**: 느린 처리, 높은 리소스 요구, 최신 기술이라 검증 부족
- **추천**: 고정확도 요구, 학술 연구, GPU 환경 보유

### librosa (2015+, 오픈소스)
- **장점**: 매우 빠른 처리, 설치/사용 간편, 의존성 최소
- **단점**: 낮은 정확도(70%), 복잡한 음악에 약함
- **추천**: 빠른 프리뷰, 교육용, 실시간 처리 필요 시
