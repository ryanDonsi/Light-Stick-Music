## 3. 파일 추가 (1분)

### 방법 1: 파일 다이얼로그
1. "📁 파일 추가" 버튼 클릭
2. MP3/WAV/FLAC 파일 선택
3. "열기" 클릭

### 예시
```
선택할 파일:
- Shape of You.mp3 (96.77 BPM - 약간 어려움)
- Dynamite.mp3 (120 BPM - 표준)
- Slow Song.mp3 (70 BPM - 느린 곡)
```

## 4. 분석 (1-2분)

1. 파일 목록에서 곡 선택
2. "🔄 분석 시작" 버튼 클릭
3. "분석 중..." → "분석 완료" 까지 대기

### 첫 분석
⏱️ 20초 ~ 3분 소요 (파일 크기에 따라)

### 이후 분석
⚡ 빠름 (메모리 캐시)

## 6. 매개변수 조정 (선택)

### 결과가 부정확하면

#### 검출 BPM이 너무 높음 (2배 오류)
```
1. Prior Center 증가 (500 → 550-600)
2. Prior STD 감소 (2.0 → 1.5)
→ 느린 곡 중심으로 조정
```

#### 검출 BPM이 너무 낮음 (1/2배 오류)
```
1. Prior Center 감소 (500 → 450)
2. Prior STD 증가 (2.0 → 2.5-3.0)
→ 빠른 곡 중심으로 조정
```

#### ODF 신호가 약함
```
1. Smooth Window 증가 (3 → 5-7)
2. Local/Global Window 증가
3. Prior STD 증가 (2.0 → 2.5+)
→ 신호 강화
```

## 7. 배치 분석 (선택)

등록된 모든 곡을 한번에 분석:

1. "배치 분석" 탭 클릭
2. "🔄 배치 분석 시작" 클릭
3. 결과 확인 (곡별 GT BPM 비교)

### 출력 예시
```
곡                  GT BPM  검출 BPM  오차율  정확도
────────────────────────────────────────────────
Shape of You        96.8    96.0      0.8%    ✅
Dynamite           120.0   119.8      0.2%    ✅
Slow Song           70.0    71.0      1.4%    ❌
...
정확도: 35/42 = 83.3%
```

## 일반적인 패턴

### 케이스 1: 현재 설정이 좋음 (35/42 정확)
```
→ 그대로 유지
→ 추가 조정 필요 없음
```

### 케이스 2: 특정 곡이 부정확함
```
1. 그 곡의 ODF 곡선 확인
2. AC/Prior/Score 그래프 분석
3. 문제 원인 파악:
   - AC 신호 약함 → 신호 처리 개선 필요
   - Prior 분포 맞지 않음 → Prior Center/STD 조정
   - 다른 lag이 더 높은 점수 → 매개변수 조정으로 해결
```

### 케이스 3: 느린 곡들이 많이 틀림
```
1. Prior Center 증가 (500 → 550-600)
2. Prior STD 증가 (2.0 → 2.5-3.0)
3. Max Beat MS 증가 (1000 → 1200)
4. 배치 분석으로 정확도 확인
```

### 케이스 4: 빠른 곡들이 많이 틀림
```
1. Prior Center 감소 (500 → 450)
2. Prior STD 감소 (2.0 → 1.5)
3. Min Beat MS 감소 (375 → 300)
4. Smooth Window 감소 (3 → 2)
5. 배치 분석으로 정확도 확인
```

## 문제해결

### Q: GUI 창이 안 뜸
```
A: 터미널 에러 확인 후 설치 가이드 참고
   python3 test_odf_engine.py 로 먼저 테스트
```

### Q: "분석 중..."에서 멈춤
```
A: 파일 크기가 크면 시간이 오래 걸림
   5분 이상이면 Ctrl+C로 중단 후 작은 파일로 테스트
```

### Q: 모든 곡이 부정확함
```
A: 매개변수를 기본값으로 리셋
   SETUP.md의 "문제해결" 섹션 참고
```

### Q: 배치 분석 결과가 이상함
```
A: 각 곡을 개별 분석해서 확인
   MADMOM_GT 필드가 등록되었는지 확인 (gui_app.py의 MADMOM_GT)
```
